##Port:

A port is a logical communication endpoint used by applications to send/receive data over a network.

IP address-> building
Port-> specific room

Example: 
192.168.1.10:80 
Means:
Device 192.168.1.10 is communicating with the HTTP service running on port 80.

##Port Ranges: 

Ports are numbered 0-65535

1. Well-Known Ports (0-1023):
   Used by common protocols.
   Example:
   HTTP
   DNS
   FTP
3. Registered Ports (1024-49151):
   Used by applications:
   My SQL
   SQL server
   RDP
5. Dynamic/Private Ports (49152-65535):
   Temporarily assigned for client connection
